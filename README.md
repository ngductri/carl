## 📌 Introduction  
This project is developed by **our team** as part of our work on the **CaRL (CARLA Reinforcement Learning)** system.  
The current repository is a **fork of the official CaRL repo** with the purpose of:

- Extending and modifying the original implementation for research needs  
- Experimenting with new Reinforcement Learning algorithms in the CARLA simulator  
- Enhancing the workflow for logging, debugging, and evaluation  
- Keeping the core structure of the original project while improving modularity and maintainability  

> ⚠️ **Note:** This is *not* the official CaRL repository.  
> It is a customized internal version tailored to our experiments.
